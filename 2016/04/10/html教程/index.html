HTML 颜色名

HTML 4.0 标准仅支持 16 种颜色名，它们是：aqua、black、blue、fuchsia、gray、green、lime、maroon、navy、olive、purple、red、silver、teal、white、yellow。

PS：如果使用其它颜色的话，就应该使用十六进制的颜色值。

重要标签

<!DOCTYPE> 声明必须是 HTML 文档的第一行，位于 标签之前。

<!DOCTYPE> 声明不是 HTML 标签；它是指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。
